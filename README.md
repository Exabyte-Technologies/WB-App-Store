# WB App Store

Web Based App Store for Mac

---

A simple web based App Store for Mac which acts as an alternative GUI for installing Mac apps.

This app uses the App Store Web website interface with additional javasript for app installation.

## Features

* Browse and search for macOS applications.
* Install and uninstall applications directly from the web interface.
* Check for updates and upgrade all installed applications with a single click.
* Clear cache and reset the app store to its initial state.
* Supports both installation and uninstallation of apps.

## Requirements

* macOS
* Electron version 39.1.2
* Node.js version 16.13.0 or higher

## Usage

1. Launch the application.
3. Browse and search for macOS applications.
4. Install or uninstall applications by clicking on the appropriate buttons.
5. Check for updates and upgrade all installed applications with a single click.
6. Clear cache and reset the app store to its initial state.
